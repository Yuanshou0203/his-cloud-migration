## 二、分级权限管控

### 📍 执行位置：.20 docker-node1

```sql
-- DBA 全权限
GRANT ALL PRIVILEGES ON his_hospital.* TO 'his_dba'@'%' IDENTIFIED BY 'HisDba@2024';

-- 只读（报表）
GRANT SELECT ON his_hospital.* TO 'his_readonly'@'%' IDENTIFIED BY 'HisRead@2024';

-- 应用账号（CRUD，不能改表）
GRANT SELECT, INSERT, UPDATE, DELETE ON his_hospital.* TO 'his_app'@'%' IDENTIFIED BY 'HisApp@2024';

-- 审计账号（只能看患者和处方）
GRANT SELECT ON his_hospital.patients TO 'his_audit'@'%' IDENTIFIED BY 'HisAudit@2024';
GRANT SELECT ON his_hospital.prescriptions TO 'his_audit'@'%';

FLUSH PRIVILEGES;
```

## 三、全量数据迁移

### 📍 执行位置：.10 docker-master（远程操控两台）

```bash
# 1. 在 .20 上导出
ssh root@docker-node1 "mysqldump -u root -proot --single-transaction --master-data=2 his_hospital > /tmp/his_backup.sql"

# 2. 传输到 .40
scp root@docker-node1:/tmp/his_backup.sql /tmp/
scp /tmp/his_backup.sql root@docker-node2:/tmp/

# 3. 在 .40 上恢复
ssh root@docker-node2 "mysql -u root -proot -e 'CREATE DATABASE IF NOT EXISTS his_hospital CHARSET utf8;' && mysql -u root -proot his_hospital < /tmp/his_backup.sql"
```

| 参数 | 作用 |
|------|------|
| --single-transaction | 一致性快照，不锁表 |
| --master-data=2 | 记录 binlog 位置 |

### 📍 验证

```bash
# 源端
ssh root@docker-node1 "mysql -u root -proot his_hospital -e 'SELECT COUNT(*) FROM patients;'"
# 10

# 目标端
ssh root@docker-node2 "mysql -u root -proot his_hospital -e 'SELECT COUNT(*) FROM patients;'"
# 10
```

两端一致，迁移成功。

## 四、增量同步（binlog）

### 📍 执行位置：.20 docker-node1（配置），.10 docker-master（执行）

**1. 在 .20 开启 binlog**：

```bash
cat >> /etc/my.cnf << 'EOF'
log-bin=mysql-bin
binlog_format=ROW
server-id=1
expire_logs_days=7
EOF
systemctl restart mariadb
```

**2. 在 .10 执行增量同步**：

```bash
# 获取 binlog 位置
ssh root@docker-node1 "mysql -u root -proot -e 'SHOW MASTER STATUS;'"

# 模拟增量数据（插入一条新患者）
ssh root@docker-node1 "mysql -u root -proot his_hospital -e \"
  INSERT INTO patients VALUES(11,'MR011','增量患者','男','1992-06-15','310101199206151244','13800138999','宝山区');
\""

# 导出增量
ssh root@docker-node1 "mysqlbinlog --start-position=<位置> /var/lib/mysql/mysql-bin.000001 > /tmp/incremental.sql"

# 传输并应用
scp root@docker-node1:/tmp/incremental.sql /tmp/
scp /tmp/incremental.sql root@docker-node2:/tmp/
ssh root@docker-node2 "mysql -u root -proot his_hospital < /tmp/incremental.sql"
```

## 五、Nginx WAF 部署

### 📍 执行位置：.40 docker-node2

**1. 安装 Nginx**：

```bash
yum install -y nginx
```

**2. WAF 配置** (`/etc/nginx/conf.d/his_waf.conf`)：

```nginx
# 限流：每 IP 30 请求/秒
limit_req_zone $binary_remote_addr zone=his_api:10m rate=30r/s;
limit_conn_zone $binary_remote_addr zone=his_conn:10m;

upstream his_backend {
    server 127.0.0.1:8080 weight=3;
}

server {
    listen 80;
    limit_req zone=his_api burst=50 nodelay;
    limit_conn his_conn 50;

    # 安全响应头
    add_header X-Frame-Options "DENY" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    # === SQL 注入拦截 ===
    set $block_sql 0;
    if ($query_string ~* "union[\s+]+select|select[\s+]+from|
        insert[\s+]+into|drop[\s+]+table|delete[\s+]+from|
        update[\s+]+.*set|--|/\*") { set $block_sql 1; }
    if ($block_sql = 1) { return 403; }

    # === XSS 拦截 ===
    set $block_xss 0;
    if ($request_uri ~* "<script[^>]*>|javascript:|onerror=|
        alert\(|eval\(|document\.cookie") { set $block_xss 1; }
    if ($block_xss = 1) { return 403; }

    # === 文件上传控制 ===
    if ($request_uri ~* "\.(php[0-9]?|jsp|asp|aspx|cgi|pl|py)$") { return 403; }
    client_max_body_size 10m;

    # === 目录遍历 ===
    if ($request_uri ~* "\.\./|\.\.\\") { return 403; }

    # 正常转发
    location / {
        proxy_pass http://his_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

**3. 启动**：

```bash
systemctl stop httpd 2>/dev/null   # 停 Apache（端口冲突）
nginx -t                            # 测试配置
systemctl start nginx
systemctl enable nginx
```
