## 第二步：.20 导出数据

仍在 .20 终端：

```bash
mysqldump -u root -proot --single-transaction --master-data=2 his_hospital > /tmp/his_backup.sql
ls -la /tmp/his_backup.sql
```
看到文件存在（几 KB）就行。

## 第三步：.20 传到 .40

仍在 .20 终端：

```bash
scp /tmp/his_backup.sql root@192.168.133.40:/tmp/
# 密码: root
```

## 第四步：登录 .40 恢复数据

打开 VMware 中 **CentOS 7-2 (192.168.133.40)** 的终端：

```bash
su -
# 密码: root
```

```bash
# 创建库
mysql -u root -proot -e "CREATE DATABASE IF NOT EXISTS his_hospital CHARSET utf8;"

mysql -u root -proot his_hospital -e "SELECT COUNT(*) FROM patients;"
```

输出 `10` 就表示迁移成功。

## 第五步：.40 部署 Nginx WAF

仍在 .40 终端：

```bash
yum install -y nginx
```

创建 WAF 配置文件：

```bash
cat > /etc/nginx/conf.d/his_waf.conf << 'EOF'
limit_req_zone $binary_remote_addr zone=his_api:10m rate=30r/s;
limit_conn_zone $binary_remote_addr zone=his_conn:10m;

upstream his_backend {
    server 127.0.0.1:8080 weight=3;
}

server {
    listen 80;
    limit_req zone=his_api burst=50 nodelay;
    limit_conn his_conn 50;

    add_header X-Frame-Options "DENY" always;
    add_header X-Content-Type-Options "nosniff" always;
    add_header X-XSS-Protection "1; mode=block" always;

    set $block_sql 0;
    if ($query_string ~* "union[\s+]+select|select[\s+]+from|insert[\s+]+into|drop[\s+]+table|delete[\s+]+from|update[\s+]+.*set|--|/\*") { set $block_sql 1; }
    if ($block_sql = 1) { return 403; }

    set $block_xss 0;
    if ($request_uri ~* "<script[^>]*>|javascript:|onerror=|alert\(|eval\(|document\.cookie") { set $block_xss 1; }
    if ($block_xss = 1) { return 403; }

    if ($request_uri ~* "\.(php[0-9]?|jsp|asp|aspx|cgi|pl|py)$") { return 403; }
    if ($request_uri ~* "\.\./|\.\.\\") { return 403; }

    location / {
        proxy_pass http://his_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF
```

```bash
# 停掉 Apache（端口冲突）
systemctl stop httpd 2>/dev/null

# 测试配置
nginx -t

# 启动
systemctl start nginx
systemctl enable nginx
```

## 第六步：在 .10 上验证 WAF

打开 VMware 中 **CentOS 7-3 (192.168.133.10)** 的终端，任意用户即可：

```bash
# SQL注入测试 → 预期 403
curl -s -w '%{http_code}\n' 'http://192.168.133.40/?id=1+union+select+1'

# XSS测试 → 预期 403
curl -s -w '%{http_code}\n' 'http://192.168.133.40/?q=<script>alert(1)</script>'

# 正常请求 → 预期 200
curl -s -w '%{http_code}\n' 'http://192.168.133.40/'
```

应该看到：`403` `403` `200`。
